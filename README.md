# Simple DAO Voting App

This is a simple blockchain project for beginners.

This project helps you:
- Connect a crypto wallet
- View proposals
- Vote on proposals
- Learn how DAO systems work

This project is good for:
- Web3 learners
- Blockchain students
- Crypto developers who want practice

---

## What You Need

- Web browser
- MetaMask wallet
- Basic JavaScript knowledge
- A deployed DAO smart contract (example)

---

## How To Run

1. Download this repository
2. Open the folder
3. Open `index.html` in your browser
4. Connect your wallet
5. Enter contract address
6. Load proposals
7. Vote

---

## Project Files

- `index.html` = Main page  
- `app.js` = Main logic  
- `config.js` = Settings  
- `wallet.js` = Wallet connect  
- `dao.js` = DAO reader  
- `ui.js` = Screen updates  
- `abi.js` = DAO contract ABI  
- `utils.js` = Helpers  
- `style.css` = Styles  
- `logger.js` = Logs  

---

## Learning Goals

- Learn how DAOs work
- Learn how on-chain voting works
- Learn how smart contracts control decisions
